# Agendador de Tarefas

## Descrição
Este programa foi desenvolvido em Python com o objetivo de simular um sistema simples de gerenciamento de tarefas.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Calcular tempo total das tarefas
- Encerrar o sistema

## Como funciona
O sistema apresenta um menu interativo ao usuário, permitindo escolher as opções disponíveis.

As tarefas são armazenadas em uma lista contendo:
- Nome da tarefa
- Tempo estimado
- Status (concluída ou não)

## Tecnologias utilizadas
- Python
- Vs Code
